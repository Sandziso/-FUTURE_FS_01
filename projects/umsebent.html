Project Overview

Umsebenti Wemaswati is a sophisticated local services marketplace designed to empower Eswatini's skilled workforce by connecting service providers directly with customers. The platform revolutionizes how local services are discovered, booked, and paid for in the digital age.

👨‍💻 My Role: Sole Full-Stack Developer

I was the sole developer responsible for the entire MERN stack application:

  • Frontend Development: Complete React.js application with responsive design
  • Backend API: Node.js/Express RESTful API with MongoDB integration
  • Real-time Features: WebSocket implementation for live messaging
  • Payment Systems: Mobile money and traditional payment integration
  • Database Design: Complex MongoDB schema with 20+ collections
  • DevOps & Deployment: Full deployment pipeline and server configuration

Technologies Used

React.js Node.js Express.js MongoDB Mongoose Socket.io JWT Authentication Mobile Money API Google Maps API Cloudinary Bootstrap 5 AWS EC2
